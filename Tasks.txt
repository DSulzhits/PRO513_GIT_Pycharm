Задача 1: Создание репозитория и выполнение коммита изменений

Ситуация:
Вы начали работу над проектом и хотите использовать Git для управления версиями. 
Вам необходимо исключить из репозитория ненужные файлы, например настройки VS Code. 
Для этого нужно создать файл .gitignore и зафиксировать его с помощью коммита.

Задача:
Создайте новый репозиторий, добавьте файл .gitignore, 
укажите в нём исключения для папок .vscode/ и .idea/ и зафиксируйте изменения в репозитории.

Шаги реализации:

Создайте новый каталог для вашего проекта.
Инициализируйте репозиторий Git с помощью команды git init.
Создайте файл .gitignore и добавьте правила.
Добавьте файл в репозиторий с помощью команды git add.
Зафиксируйте изменения с помощью команды git commit.

Задача 2: Добавляем файл README.md в существующий репозиторий

Ситуация:
Вы начали работу над проектом, и у вас уже создан репозиторий с файлами.
Вам нужно добавить файл README.md, чтобы описать ваш проект.

Задача:

Создайте файл README.md с названием проекта и зафиксируйте его в репозитории.

Шаги реализации:

Перейдите в каталог вашего проекта.
Создайте файл README.md.
Добавьте название проекта в файл README.md.
Добавьте файл в область индексирования.
Зафиксируйте изменения с помощью коммита.
