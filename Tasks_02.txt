Задача 1: Создание репозитория, добавление изменений и работа с ветками

Ситуация:

В процессе работы над проектом вы решили использовать ветки для разработки новых функций.
У вас уже есть репозиторий my-project, где создан файл README.md и настроен .gitignore.
Теперь вам нужно создать новую ветку, в которой вы будете проводить эксперименты.
Добавьте новый файл в созданной ветке и выполните слияние.

Задача:

Создайте новую ветку, внесите изменения в этой ветке, а затем выполните слияние с основной веткой.

Шаги реализации:

Перейдите в репозиторий my-project.
Создайте новую ветку.
Внесите изменения в новую ветку.
Выполните коммит.
Выполните слияние ветки с основной.


Задача 2: Создание репозитория, добавление функции и работа с ветками

Ситуация:

В процессе работы над проектом вы решили создать новую функцию для приветствия
пользователя и использовать ветки для разработки новых функций.
У вас уже есть репозиторий my-project, где создан файл README.md и настроен .gitignore.
Теперь вам нужно сформировать новую ветку для экспериментов,
добавить в ней функцию приветствия и выполнить слияние изменений с основной веткой.

Задача:

Создайте новую ветку, добавьте функцию приветствия пользователя в этой ветке,
а затем выполните слияние изменений с основной веткой.
